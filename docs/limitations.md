# Known limitations

- The durable worker boundary is represented, but local submissions process inline for a reliable one-command demo.
- MinIO is provisioned locally; the current filesystem adapter is the active storage implementation.
- Webhook delivery outcomes are simulated and do not send arbitrary network traffic.
- No external artwork availability check is performed.
- Analytics use deliberately seeded history and must not be read as customer evidence.
- PDF reports and public deployment are intentionally omitted.
